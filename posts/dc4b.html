基于CentOS7的Hexo博客搭建
发表于 2019-08-18| 更新于 2020-04-12|博客部署CentOS
阅读量:

1.环境搭建

安装Node.js

1)更新系统 sudo yum update -y 2)安装EPEL库 sudo yum install epel-release -y 3)安装Node.js sudo yum install nodejs 4)检查安装


5)安装Git sudo apt-get install git

2.安装Hexo博客框架

1)利用npm安装淘宝源(此步可以省略) npm install -g cnpm --registry https://registry.npm.taobao.org 2)安装Hexo npm install -g hexo-cli 3)检查安装 hexo -v

3.开始使用Hexo搭建博客

这里直接使用Hexo官网提供的命令


当看到Start blogging with Hexo! 时，说明搭建成功了 这时访问localhost:4000就可以看到你生成的博客了。

4.将hexo部署到GitHub

此处借鉴hexo史上最全搭建教程
1.在github上搭建仓库，仓库名于用户名相同
2.生成SSH添加到GitHub 参考上述链接即可
