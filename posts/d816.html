利用LaTex语法写化学式_化学方程式
发表于 2020-03-16| 更新于 2020-04-03|博客部署LaTeX
阅读量:

上篇文章中已经有使用过化学方程式，这次在这里集中做一下技术总结

化学式

对于任意化学式，都可以通过LaTeX语法书写，但结果需要mathjax等的渲染
语法如下


其中”_”后跟下标，”^”后写上标
显示效果如下
$SO^{2-}_4$
$NH^+_4$

注：化合价目前未找到怎么标

方法：元素_下标^上标 （元素后上下标若不为一个整体则需用{}括起）

表达式

显示效果如下
$Fe + CuSO_4 \xrightarrow{} Cu + FeSO_4$
$2KClO_3 \xrightarrow[\Delta]{MnO_2} 2KCl + 3O_2\uparrow$

方法 A+B \xrightarrow[反应条件下方]{反应条件上方} C + D

化学方程式

显示效果如下
$Ca(OH)_2 + CO_2 \overset{}= CaCO_3\downarrow + H_2O$
$2KMnO_4 \triangleq K_2MnO_4 + MnO_2 O_2\uparrow$
$C_2H_5OH +3O_2 \overset{点燃}= CO_2 +3H_2O$

方法：A + B \overset{反应条件}= C + D
